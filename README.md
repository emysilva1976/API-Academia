# 🏋️ API Academia

API REST desenvolvida em Java com Spring Boot para gerenciamento de alunos de academia.

## 🚀 Tecnologias

- Java 21
- Spring Boot
- Spring Security
- JWT
- Spring Data JPA
- MySQL
- Railway
- Render
- Docker
- Maven
- Swagger/OpenAPI

## 📌 Funcionalidades

- Cadastro de usuários
- Login com JWT
- Cadastro de alunos
- Atualização de alunos
- Exclusão de alunos
- Consulta de alunos
- Documentação Swagger

## 🔒 Segurança

A API utiliza autenticação JWT.

Existem dois perfis:

- ADMIN
- USER
